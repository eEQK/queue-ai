Emergency Room Queue Management API

Phase 2 - Core API Infrastructure

Available Endpoints:

GET /api/health - API health check
GET /api/queue/current - Current queue status
GET /api/queue/status - Queue status with trends and alerts
GET /api/queue/history/:hours - Historical queue data
GET /api/queue/analytics - Queue analytics
GET /api/iot/sensors - Sensor summaries
GET /api/iot/system-health - IoT system health
GET /api/dashboard/overview - Dashboard overview data
GET /api/dashboard/real-time - Real-time dashboard data

Note: Phase 3 will add ML prediction endpoints
